# Contributing

Thanks for helping improve Transient Recall.

## Contribution Guidelines

- Keep contributions public-safe.
- Do not commit secrets, private URLs, internal identifiers, or proprietary internals.
- Open an issue before major architecture or policy changes.
- Keep pull requests small and focused.

## Pull Request Checklist

- Problem statement and rationale included
- README/docs updated when behavior changes
- Security-sensitive content excluded
